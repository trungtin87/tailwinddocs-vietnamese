## Ví dụ

### Tăng tốc phần cứng

Nếu transition của bạn hoạt động tốt hơn khi được render bởi GPU thay vì CPU, bạn có thể bắt buộc tăng tốc phần cứng bằng cách thêm utility `transform-gpu`:

```html
<!-- [!code classes:transform-gpu] -->
<div class="scale-150 transform-gpu">
  <!-- ... -->
</div>
```

Sử dụng utility `transform-cpu` để buộc mọi thứ quay lại CPU nếu bạn cần hoàn tác điều này có điều kiện.

### Xóa các biến đổi

Sử dụng utility `transform-none` để xóa tất cả các biến đổi trên một phần tử cùng một lúc:

```html
<!-- [!code classes:transform-none] -->
<div class="skew-y-3 md:transform-none">
  <!-- ... -->
</div>
```
