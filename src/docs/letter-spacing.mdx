### Sử dụng giá trị âm

Sử dụng giá trị âm không có nhiều ý nghĩa với thang đo khoảng cách chữ cái được đặt tên mà Tailwind bao gồm sẵn, nhưng nếu bạn đã tùy chỉnh thang đo của mình để sử dụng số thì nó có thể hữu ích:

```css
@theme {
  --tracking-1: 0em;
  --tracking-2: 0.025em;
  --tracking-3: 0.05em;
  --tracking-4: 0.1em;
}
```

Để sử dụng giá trị khoảng cách chữ cái âm, hãy thêm dấu gạch ngang trước tên class để chuyển đổi nó thành giá trị âm:

```html
<!-- [!code classes:-tracking-2] -->
<p class="-tracking-2">The quick brown fox ...</p>
```
